我们应该做什么：

    **TCP-in-IP-in-Ethernet.（我们将要实现的）** 网络接口将出站 IP 数据报转换为链路层（以太网）帧，反之亦然。代替 Linux 提供的 TUN 设备接口，这样 Linux 内核可以直接获得并处理以太网帧。

    1. 出站 IP -> 以太帧

    2. 以太帧 -> 入站ip

    3. 以太帧 -> 入站ip包 -> 出战ip包 -> 以太帧（转发)
